# Cloudflare Configuration

This directory contains Cloudflare configurations for DNS, WAF, and edge security.

## Services

- DNS management
- Web Application Firewall (WAF)
- Bot protection
- Rate limiting
- DDoS protection

## Security Headers

- HSTS enabled
- CSP with nonce
- No wildcard CORS
- Frame denial
- MIME sniff prevention

## Setup

Configuration is managed through Cloudflare dashboard and Terraform (future).
